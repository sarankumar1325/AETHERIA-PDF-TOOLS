v0pdftools is a high-performance, browser-based suite of PDF manipulation tools designed with a focus on privacy and a premium Heritage Sandal aesthetic. All processing is executed strictly on the client side, ensuring that your sensitive documents never leave your local environment.

## Key Features

- **Merge Intelligence**: Combine multiple PDF files into a single document.
- **Split Intelligence**: Divide a PDF into individual pages or sections.
- **Precision Rotation**: Correct document orientation with mathematical accuracy.
- **Selective Deletion**: Remove specific pages or ranges from any PDF.
- **AES-256 Encryption**: Apply industrial-grade password protection locally.
- **Vision Synthesis**: Convert images (JPG, PNG, WebP) into professional PDFs.
- **Neural HTML Renderer**: Transform raw HTML markup into PDF documents.
- **Identity Layering**: Imprint documents with secure text watermarks.
- **Pixel Extraction**: Export PDF pages as high-fidelity PNG images.

## Technology Stack

- **Runtime**: Bun
- **Framework**: React
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion, GSAP
- **PDF Engine**: pdf-lib, jspdf, pdfjs-dist
- **State Management**: Zustand

## Privacy and Security

v0pdftools is architected for total isolation. No data is transmitted to external servers. By utilizing modern web standards and client-side cryptographic primitives, we guarantee that your information remains under your exclusive control.

## Development

To initialize the development environment:

```bash
bun install
```

To launch the local server:

```bash
bun dev
```

To build for production:

```bash
bun run build.ts
```
